Installation

System Requirements

  • 64-bit processor (x86_64)
  • 2GB RAM minimum (4GB+ recommended)
  • 20GB free disk space
  • UEFI or Legacy BIOS support

Download & Setup

  1. Download Xray_OS ISO from SourceForge
  2. Create bootable USB using Rufus (Windows), Balena Etcher, or Ventoy
  3. Boot from USB and follow the installation wizard
  4. The installer will automatically detect and configure GPU drivers
  5. Reboot and enjoy your new Xray_OS system

Tolito is Xray_OS's custom AUR helper that builds and downloads packages from curated PKGBUILDs, with automatic fallback to the standard AUR.

Basic Commands

# Install a package
tolito -S package_name
# Remove a package
tolito -R package_name
# Query package info
tolito -Qi package_name
# Clean package cache
tolito clean

Key Features

  • Curated PKGBUILDs that fix broken AUR packages
  • Easy DaVinci Resolve installation without errors
  • Automatic fallback to standard AUR
  • Pacman-compatible syntax
